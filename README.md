## If you use Yarn:
### Install dependencies first
```
    yarn add
```
### Run in window:
```
    yarn tauri dev
```
### Run in browser:
```
    yarn dev
```
### Build project:
```
    yarn tauri build
```

&nbsp;

## If you use npm:
### Install dependencies first
```
    npm install
```
### Run in window:
```
    npm run tauri dev
```
### Run in browser:
```
    npm run dev
```
### Build project:
```
    npm run tauri build
```
